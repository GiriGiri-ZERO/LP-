**プロジェクト名:** AI Copy Studio  
**バージョン:** 1.0  
**作成日:** 2026-04-22  
**ステータス:** ドラフト

---

## 1. プロジェクト概要

### 1.1 目的

LP・Webサイト・セールスレターなどのコピーをAIで自動生成し、Canvaライクなビジュアルエディタで細部まで修正・カスタマイズできるWebアプリケーションを提供する。

コピーライターの専門知識がないユーザーでも、プロ品質のマーケティングコピーを短時間で作成できるようにする。

### 1.2 背景・課題

| 現状の課題 | 解決アプローチ |
|---|---|
| 優秀なコピーライターへの依頼コストが高い | AIによる高品質コピーの自動生成 |
| コピー修正のたびにライターとのやり取りが発生 | リアルタイムなセルフ編集環境 |
| テンプレートツールでは表現の自由度が低い | ブロックベースの柔軟なエディタ |
| A/Bテスト用バリエーション作成が手間 | AIによる複数バリエーション生成 |

### 1.3 ターゲットユーザー

- **プライマリ:** 中小企業のマーケター・事業主（コピーライティング専門知識が浅い）
- **セカンダリ:** フリーランスコピーライター・Webデザイナー（作業効率化目的）
- **ターシャリ:** マーケティングエージェンシー（クライアント対応の効率化）

## 3. 非機能要件

### 3.1 パフォーマンス

| 要件 | 目標値 |
|---|---|
| AIコピー生成のレスポンス | 初回トークン表示まで3秒以内（ストリーミング） |
| エディタ操作のレスポンス | 100ms以内 |
| ページ初期ロード | LCP 2.5秒以内 |
| 同時接続ユーザー | 1,000ユーザー（フェーズ1） |

### 3.2 可用性・信頼性

- サービス稼働率: 99.5%以上（月間）
- データバックアップ: 1日1回、30日間保持
- AIサービス障害時のフォールバック対応

### 3.3 セキュリティ

- 通信の全HTTPS化
- ユーザーデータの暗号化保存
- APIキーのサーバーサイド管理（クライアント露出なし）
- OWASP Top10対策
- GDPRおよび個人情報保護法への準拠

### 3.4 アクセシビリティ

- WCAG 2.1 AA準拠
- キーボード操作対応
- スクリーンリーダー対応（エディタ周辺は除く）

### 3.5 対応ブラウザ・デバイス

- Chrome / Edge / Safari / Firefox 最新2バージョン
- デバイス: PC優先（タブレットはビューアーとして対応）

## 4. ユーザーストーリー

### Epic 1: コピー生成

```
As a 事業主
I want to 商品情報を入力するだけでLPコピー全体を生成できる
So that コピーライターに依頼せずに質の高いLPを作れる
```

```
As a マーケター
I want to 生成したコピーのセクションを個別に再生成できる
So that 気に入った部分を残しつつ改善できる
```

### Epic 2: エディタ操作

```
As a ユーザー
I want to テキストをクリックしてその場で編集できる
So that 直感的にコピーを修正できる
```

```
As a ユーザー
I want to ブロックをドラッグ&ドロップで並び替えられる
So that ページ構成を素早く調整できる
```

### Epic 3: エクスポート

```
As a Webデザイナー
I want to 完成したコピーをHTMLで書き出せる
So that 既存のサイト制作フローに組み込める
```

## 5. 制約・前提条件

### 5.1 技術制約

- AI生成にはAnthropicのClaude APIを使用
- フロントエンドはWebアプリ（モバイルアプリはスコープ外）
- 初期フェーズでは日本語コピー生成に特化

### 5.2 ビジネス制約

- フェーズ1のMVPリリース目標: 着手から4ヶ月以内
- 初期開発チーム: エンジニア2〜3名 + デザイナー1名

### 5.3 スコープ外（フェーズ1）

- 動画・アニメーションの生成・編集
- メール配信・広告配信との直接連携
- 多言語対応（英語等）
- モバイルアプリ

## 6. リスクと対策

| リスク | 影響度 | 発生確率 | 対策 |
|---|---|---|---|
| AI生成コピーの品質ばらつき | 高 | 中 | プロンプトエンジニアリングの最適化、ユーザーフィードバックループ |
| Claude APIのレート制限・障害 | 高 | 低 | リトライ機構、キューイング、複数モデルフォールバック |
| エディタのブラウザ互換性問題 | 中 | 中 | 主要ブラウザでの自動テスト |
| 著作権・コンテンツポリシー問題 | 高 | 低 | 利用規約整備、生成コンテンツのガイドライン |

## 8. 用語定義

| 用語 | 定義 |
|---|---|
| コピー | マーケティング目的で書かれたテキストコンテンツ全般 |
| LP | ランディングページ。広告やメールからの流入を受けるページ |
| セールスレター | 商品・サービスの購入を促す長文形式のコピー |
| ブロック | エディタ上の編集単位。ヘッドライン、ボディ、CTAなど |
| バリエーション | 同一目的で表現を変えた複数のコピー案 |
